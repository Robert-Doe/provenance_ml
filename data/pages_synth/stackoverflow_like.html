How do I debounce an input in vanilla JavaScript?

I want to delay an API call until typing stops for ~300ms. What’s a minimal implementation?

function debounce(fn, wait=300){
  let t;
  return (...args) => {
    clearTimeout(t);
    t = setTimeout(() => fn(...args), wait);
  };
}

With leading invocation:

function debounceLeading(fn, wait=300){
  let t, called=false;
  return (...args) => {
    if (!called) { fn(...args); called=true; }
    clearTimeout(t);
    t=setTimeout(()=>{ called=false; }, wait);
  };
}
improves perceived responsiveness
